SCADA Config Git Manager

Приложение на Flutter для контроля версий текстовых конфигураций SCADA с помощью Git.

Позволяет инициализировать репозиторий в заданной папке, видеть информацию о последнем коммите (сообщение, хеш, дата) и создавать новые коммиты с автоматическим увеличением версии.

Использование

Выберите папку с конфигурациями SCADA.

Приложение инициализирует Git-репозиторий (если его нет).

На экране отображается информация о последнем коммите.

Если есть изменения, нажмите кнопку «Создать коммит», чтобы зафиксировать новые изменения с увеличением версии.
